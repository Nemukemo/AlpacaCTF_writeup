## 問題理解
この問題は下記の流れで実行されている
1. 文字列の入力
2. 入力した文字列を大文字の文字を小文字へ、小文字の文字を大文字へする処理を2回実行する
3. 処理を実行した文字列と入力された文字列が一致したかどうかで処理が分かれる
   - 一致した場合：処理が終了
   - 一致しない場合：Flagを取得できる
一見すると大文字小文字を2回変えるためFalseにしかならなそうである 
## 解法
PythonはUnicodeを使用しており、unicodeには多くの言語の文字列がありそれにより**大文字小文字変換ができない、変換したら文字が増える**といった仕様がある言語があったりします。それを送り込めばOKです。
調べてその文字列を使うでもいいですが、今回はgeminiくんが二重変換時に文字列が異なる文字を調査するコードを作ってくれたので、それを用いました
```python
# 0から65535（基本多言語面）の範囲で探す例
sum = ""
for i in range(65536):
    char = chr(i)
    if char != char.swapcase().swapcase():
        sum += char
        print(f"見つけた！ 文字: {char}, コードポイント: {i}")
print(sum)
```
一文字一文字検証したら日が暮れるので出力結果をひとつの文字列としてペイロードにしました。あとは自分の目で確かめてね💛
